(\cos\theta + i\sin\theta)^n = \cos(n\theta) + i\sin(n\theta)

Next, prove the DeMoivre formula for negative exponent. We need two helper lemmas. We first show that \cos x + i \sin x ≠ 0 by reduction to \cos ^2 x + \sin ^2x =1.

/-- Helper lemma 1 cos x + (sin x)*I ≠ 0 -/ lemma cos_add_sin_mul_I_ne_zero (x : ℝ) : (cos (x) + sin (x) * I : ℂ) ≠ 0 := x:ℝ⊢ cos ↑x + sin ↑x * I ≠ 0 x:ℝ⊢ (cos ↑x).re = 0 → ¬(sin ↑x).re = 0; x:ℝ⊢ Real.cos x = 0 → ¬Real.sin x = 0 intro h1 x:ℝh1:Real.cos x = 0h2:Real.sin x = 0⊢ False All goals completed! 🐙 ;

Using the first helper lemma, compute the inverse of \cos x + i \sin x.

/-- Helper lemma 2 The inverse of (cos (x) + sin (x) *I) equals cos (-x) + sin (-x) *I -/ lemma DeMoivre_helper (x : ℝ) : 1 / (cos (x) + sin (x) * I) = cos (-x) + sin (-x) * I := x:ℝ⊢ 1 / (cos ↑x + sin ↑x * I) = cos (-↑x) + sin (-↑x) * I x:ℝ⊢ cos ↑x + sin ↑x * I ≠ 0x:ℝ⊢ 1 = (cos (-↑x) + sin (-↑x) * I) * (cos ↑x + sin ↑x * I) x:ℝ⊢ cos ↑x + sin ↑x * I ≠ 0 All goals completed! 🐙 x:ℝ⊢ 1 = (cos (-↑x) + sin (-↑x) * I) * (cos ↑x + sin ↑x * I) x:ℝ⊢ 1 = Real.cos x * Real.cos x + Real.sin x * Real.sin x ∧ 0 = Real.cos x * Real.sin x + -(Real.sin x * Real.cos x) exact ⟨x:ℝ⊢ 1 = Real.cos x * Real.cos x + Real.sin x * Real.sin x All goals completed! 🐙, x:ℝ⊢ 0 = Real.cos x * Real.sin x + -(Real.sin x * Real.cos x) All goals completed! 🐙⟩

Prove that DeMoivre formula holds for negative integer n.

/-- DeMoivre formula for negative integer -/ theorem DeMoivre_neg_exp (x : ℝ) (n : ℕ) : 1/(cos (x) + sin (x) *I)^n = cos (-n*x) + (sin (-n*x))*I := x:ℝn:ℕ⊢ 1 / (cos ↑x + sin ↑x * I) ^ n = cos (-↑n * ↑x) + sin (-↑n * ↑x) * I have hD : 1/(cos (x) + sin (x) *I)^n = cos (n*(-x)) + sin (n*(-x)) * I := x:ℝn:ℕ⊢ 1 / (cos ↑x + sin ↑x * I) ^ n = cos (-↑n * ↑x) + sin (-↑n * ↑x) * I x:ℝn:ℕ⊢ (1 / (cos ↑x + sin ↑x * I)) ^ n = cos (↑n * -↑x) + sin (↑n * -↑x) * I x:ℝn:ℕ⊢ (cos (-↑x) + sin (-↑x) * I) ^ n = cos (↑n * -↑x) + sin (↑n * -↑x) * I All goals completed! 🐙 calc 1/(cos (x) + sin (x) *I)^n _= cos (n*(-x)) + sin (n*(-x)) * I := x:ℝn:ℕhD:1 / (cos ↑x + sin ↑x * I) ^ n = cos (↑n * -↑x) + sin (↑n * -↑x) * I⊢ 1 / (cos ↑x + sin ↑x * I) ^ n = cos (↑n * -↑x) + sin (↑n * -↑x) * I All goals completed! 🐙 _ = cos (-n*x) + sin (-n*x) * I := x:ℝn:ℕhD:1 / (cos ↑x + sin ↑x * I) ^ n = cos (↑n * -↑x) + sin (↑n * -↑x) * I⊢ cos (↑n * -↑x) + sin (↑n * -↑x) * I = cos (-↑n * ↑x) + sin (-↑n * ↑x) * I All goals completed! 🐙

Finally, we prove DeMoivre's formula for all integers.
