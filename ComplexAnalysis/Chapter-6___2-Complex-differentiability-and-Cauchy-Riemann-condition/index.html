import Mathlib.Tactic
import Mathlib.Data.Complex.Basic

Definition. Let f : ℂ → ℂ be a complex function defined in a domain D. Given a point z₀ ∈ D, we say that f is complex differentiable at z₀ if there exists a complex constant w₀ such that

f(z₀ + Δz) ≈ f(z₀) + w₀ · Δz

in the sense that the error is o(Δz) as Δz → 0. The constant w₀ is called the derivative of f at z₀.

In Mathlib, this is captured by HasDerivAt f w₀ z₀, which asserts that:

‖f(z₀ + h) - f(z₀) - w₀ • h‖ / ‖h‖ → 0 as h → 0.

We prove the equivalence with the classical limit definition: f is complex differentiable at z₀ with derivative w₀ if and only if lim_{h → 0} (f(z₀ + h) - f(z₀)) / h = w₀.
